## Installation
clone the project
```bash
git clone https://github.com/renaldi-oss/smart.git  
```
```bash
composer install
```
```bash
cp .env.example .env
```
```bash
php artisan key:generate
```
```bash
php artisan migrate:fresh --seed
```
```bash
php artisan serve
```
